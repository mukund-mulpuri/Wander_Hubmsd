## Placeholder Images:
- `placeholder.svg` - Generic SVG placeholder images (automatically used when specific images are not available)
  - Destinations: Gray placeholder with "Destination Image" text
  - Hotels: Yellow/orange placeholder with "Hotel Image" text

## Image Requirements:
- **Format**: JPG or PNG
- **Resolution**: Minimum 800x600px (recommended 1200x800px)
- **Aspect Ratio**: 4:3 or 16:10 works best
- **File Size**: Keep under 500KB for faster loading

## Instructions:
1. Create the images with the exact filenames listed above
2. Place destination images in the `/destinations/` folder
3. Place hotel images in the `/hotels/` folder
4. If you don't have a specific image, the system will automatically use placeholder.svg
5. Make sure image names match exactly (case-sensitive)
6. Replace the placeholder SVG files with actual images when you have them
